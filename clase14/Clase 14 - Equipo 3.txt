Investigar y realizar en la sintaxis del lenguaje dado, la siguiente operación matemática:
○ x = 4 
○ y = 5 
○ z = x + y 
○ mostrar por pantalla z 

GO:
package main
import "fmt"
func main() { 
    x:=4
    y:=5
    z:=x+y

C++:
#include <iostream>
int main() {
    int x=4;
    int y=5;
    int z=x+y;
    std::cout <<"El resultado es: " << z << std::endl;
    return 0;
}
