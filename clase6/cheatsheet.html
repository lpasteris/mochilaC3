• Git init:Crear un nuevo repositorio local
  • Git add . Permite agregar todos los archivos
  • git status permite ver el estatus de los archivos
  • git branch permite crear una rama nueva
  • git push origin main permite enviar archivos a nuestra nube en github
  • git pull origin main permite traer la ultima version del proyecto
  • git merge permite unir una rama con el main
  • git config user.mail o user.name para logearse con la pagina de github
  • mv permite mover archivos
  • touch permite crear archivos
  • nano permite modificar archivos desde la terminal
  • cd permite entrar en las carpetas
  • cd .. permite salirse de las carpetas
