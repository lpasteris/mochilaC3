Recordatorio Clase 6

  1. Git init:Crear un nuevo repositorio local
  2. Git add . Permite agregar todos los archivos
  3. git status permite ver el estatus de los archivos
  4. git branch permite crear una rama nueva
  5. git push origin main permite enviar archivos a nuestra nube en github
  6. git pull origin main permite traer la ultima version del proyecto
  7. git merge permite unir una rama con el main
  8. git config user.mail o user.name para logearse con la pagina de github
  9. mv permite mover archivos
  10. touch permite crear archivos
  11. nano permite modificar archivos desde la terminal
  12. cd permite entrar en las carpetas
  13. cd .. permite salirse de las carpetas
